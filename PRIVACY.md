# Cotab Privacy Policy

Last updated: May 11, 2026

Cotab is a Chrome extension that replaces the new tab page with a focused tab management dashboard.

## Data Cotab Handles

Cotab handles limited web history-related tab metadata needed to provide its core features, including:

- tab title
- tab URL
- domain
- favicon URL
- tab ID and window ID
- activation count
- last accessed time
- staged tab records
- theme and language preferences

## How Cotab Uses Data

Cotab uses this data only to:

- display currently open tabs
- rank active tabs by usage
- stage tabs for later use
- restore staged tabs
- close tabs at the user's request
- remember local interface preferences

## Data Storage

Cotab stores data locally on the user's device using `chrome.storage.local`.

Cotab does not operate a backend server and does not transmit this data to the developer or to third parties.

## Data Sharing

Cotab does not sell, rent, share, or transfer user data to third parties.

## Remote Code

Cotab does not execute remote code. All executable extension code is bundled in the submitted Chrome extension package.

## Analytics and Advertising

Cotab does not use analytics, advertising, tracking pixels, or behavioral profiling.

## Website Content

Cotab does not read webpage body content, form inputs, messages, screenshots, or page DOM content.

## Data Deletion

Users can delete Cotab's locally stored data by removing the extension from Chrome or clearing the extension's site/app data.
